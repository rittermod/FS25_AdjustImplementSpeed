# Changelog

## 0.0.2.0 (Experiment)
- Added reset keybind (RightShift+3) to restore default working speed and disable antigravity
- Fixed speed display to respect player's unit preference (km/h or mph)

## 0.0.1.0 (Experiment)
- Added dynamic working speed adjustment for attached implements (RightShift+1/2)
- Added antigravity mode to bypass physics-based speed penalties (RightShift+0)
- Added F1 help panel status line showing current working speed and antigravity state
- Keybindings appear automatically when vehicle has speed-limited implements attached
- Works while AI worker is driving
